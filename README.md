Integers and Decimals breakup and restore for addition

and a Division by 2 
with a set of number to achieve the same result set 

//

acherontas@rpifive:~/codeblocks/stepping_add_arm64/bin/Debug $ ./stepping_add_arm64 1225 5672.79 12
enter first number then enter second number then enter a precision
decimal is zero 

of 1225 + 5672.79 ::== 6897.79

of 1225 + 5672 ::== 6897 and  1225 + 0.79 ::== 1225.79

of 0 + 5672 ::== 5672 and  0 + 0.79 ::== 0.79

of 6897 + 1225.79 + 5672 + 0.79  ::== 13795.58 --> 13795.58 /  2  ::== 6897.79

of 6897 - 1225.79 ::== 5671.21

of 5672 - 0.79 ::== 5671.21

of 5672 - 6897 ::== -1225

of 0.79 - 1225.79 ::== -1225
